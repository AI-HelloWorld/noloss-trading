## 🔧 如果遇到问题

### 问题1: 价格显示为0

**原因**: 后端未启动或刚启动

**解决**: 
1. 确认后端正在运行
2. 等待5-10秒让数据加载
3. 手动点击刷新按钮

### 问题2: 看不到市场行情标签

**原因**: 前端未更新

**解决**:
```bash
cd frontend
npm run dev
```

### 问题3: 后端启动报错

**检查**:
1. Python版本 ≥ 3.10
2. 依赖已安装: `pip install -r requirements.txt`
3. 端口8001是否被占用: `netstat -ano | findstr :8001`

## 📊 技术细节

### 数据更新频率
- **市场数据**: 每5分钟自动更新
- **交易周期**: 每1小时（可配置）
- **数据广播**: 每10分钟（可配置）

### 数据来源
- **模拟模式**: 使用mock_market_data.py生成
- **真实模式**: 调用AsterDEX API
- **当前**: 模拟模式，129个交易对

### API端点
```
GET  /api/market-data          # 获取市场数据
POST /api/market-data/refresh  # 手动刷新
GET  /api/status               # 系统状态
GET  /api/team                 # AI团队状态
```
