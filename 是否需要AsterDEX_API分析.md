## 💰 AsterDEX API 详情

### API类型

**AsterDEX提供的API是免费的：**

```
API类型: REST API
费用: 免费
权限: 查询、交易（不含提现）
限制: 
  - 查询限制: 1200次/分钟
  - 交易限制: 50次/10秒
  - Websocket: 10个连接
```

### 如何获取API密钥

**步骤：**
1. 注册AsterDEX账户：https://www.asterdex.com
2. 完成KYC认证（可选，取决于交易额）
3. 进入API管理页面
4. 创建API密钥
5. 设置权限（只开启交易，不开启提现）
6. 保存密钥到 `.env` 文件

**安全设置：**
```
✅ 开启权限: 读取账户信息、交易
❌ 禁用权限: 提现、转账
✅ IP白名单: 设置您的服务器IP
✅ 密钥有效期: 建议90天后更新
```

### API密钥配置

在 `.env` 文件中：
```env
# AsterDEX真实交易API
ASTER_DEX_API_KEY=your_api_key_here
ASTER_DEX_API_SECRET=your_api_secret_here
```
